# i3wm_configs
My i3 windows manager configuration files
for Linux Mint 18.3 <br />


# i3.config

KEY DEFINITIONS

$Mod = WINDOWS key or Super key or Mod4 <br />
mod = ALT key  <br />
Control = CTRL key  <br />
Shift = SHIFT key  <br />
Escape = ESCAPE key  <br />
Return = ENTER or RETURN key  <br />
Pause = PAUSE key  <br />
Print = PRINT key  <br />
Tab = TAB key  <br />
